This is a highcharts demo 1 2, as this seems popular graph library. Highcharts is another library that builds SVG graphs, so they are fairly easy to style and encourage to be responsive. I made a page similar to this at Hoptroff, but that is company source. This only took a short while; I am focussing on branding things not the data; which is why I used data from the documentation.
As I have quickly built this, I don't perceive risk is starting a role that uses this graph library rather than the one I used Hoptroff. I don't think this code was hard or impressive.
As far as I can see all the animation effects are just applying the data that you already have onto the screen 3 4 5. Therefore in order to do useful updates a traditional solution using setInterval, or another event stream will be needed. This now has colour blend support, Highcharts integrates the LinearGradient tag 6. The visible text isn't copied and pasted from a German language website; I typed the new labels directly, and ran a spell check.

Pls watch for more than 5s Maybe I should add a count-down timer.
