Alternative title data plotters. I ~ in 2012/3 ~ needed a schema visualisation tool for databases at work (article has been updated, so it is still published). The main DB is established, and has lots of concurrent use. I would like a tool to convert an SQL schema to a picture; without needing a human to draw it. There are many tools that are just outputs for a human; it seems that that tool market thinks everyone wants to draw diagrams by hand. When it was current, TogetherJ was a useful tool.

The most common tool for the search term “db visualiser” is dbvisualiser. This isn't what I'm after, in that you can only see one table at once. The output is similar to sql-workbench which is what the DBA uses. The CMS team tend towards SQL developer 1, which I find slow to solve problems with. If this tool has diagramming or rasterisation features I haven't found them. UPDATE 2023: Whilst looking for something, I see this chart of tools that can build a diagram from a schema 2

As my 2013 employer is using storeproc, I fed this tool plsql-dev back to them, it seems like Eclipse, only for DB. As a comparative tool sqlcat, which is run by microsoft; and after 2015 has been cancelled. Sql-cat seems to publish helpful information 3 4 ~ warning these are docx files, rather than pages.

SQLFairy is described as SQLFairy. It requires to be fed a schema file, and produces a static image. Structures shouldn't change that often, so this isn't a concern.

Update 2016: I was sent this link pulpo, its freemium service. As its not OSS, please read the terms before committing to anything, but the initial spec look quite useful.

UPDATE 2020: @shadowcat_mst has published another tool and was talking about it. This is in Perl; the cover notes state DB->DB transform and DB->graphic transform git CPAN.

UPDATE 2022: I am updating all the HTTP links on this site. Previously I had a link to an article with many tools; but that site is gone. I find dbeaver but I think that is a SQL client rather than a SQL visualiser. DrawSQL 5 6 is a commercial tool that supports import of SQL and export of visualisations. It is described on 7. A second option is SmartDraw 8 9, this ingests a CSV export of the schema; and diagrams it.

There was a tool run by Ausberg Universität called “Mondrian”. This seems to have been bought by Pentaho, a business unit of Hitachi. Current link to Legacy docs for the first tool; but they state use Pentaho instead. This looks like a useful tool, but not relevant. It's complex data visualiser, but doesn't seem relevant to DB schema.
