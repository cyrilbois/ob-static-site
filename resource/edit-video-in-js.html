  • Suspect you do not want to do real editing on a phone in JS, as it will make the battery flat very quickly. The phone will have native libraries for making video; and all the phones I have seen allow some basic adjustments whilst writing the data to disk.
  • Basic browser support to play video 1 is quite good this doesn't edit anything. There is a Video element in HTML5 2 to allow supporting a variety of formats concurrently.
  • Video players not editors are easy to make, and are present for each JS framework 3. There is some support for editing a video frame via a HTML Canvas 4; but this seems like “Powerpoint-style design” rather than effects that could be used for a film.
